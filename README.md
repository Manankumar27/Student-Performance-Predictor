# 🎓 Student Performance Predictor

## 📌 Overview

Student Performance Predictor is a Machine Learning web application built using Python, Scikit-Learn, and Flask.

The system predicts student academic performance based on:

* Age
* GPA
* Study Time Weekly
* Absences
* Parental Support
* Tutoring
* Extracurricular Activities

The application provides predictions through an interactive web interface.

---

## 🚀 Features

* Student Performance Prediction
* Random Forest Classifier
* Scikit-Learn Pipeline
* Hyperparameter Tuning using GridSearchCV
* Flask Web Application
* Bootstrap User Interface
* Model Persistence using Joblib
